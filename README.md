# TotemZOO Bot

Telegram-бот викторина Московского зоопарка. Определяет тотемное животное пользователя и рассказывает о программе опеки.

## Функционал
- Викторина с подсчётом баллов
- Определение тотемного животного с фото и описанием
- Информация о программе опеки
- Обратная связь с зоопарком
- Система состояний (FSM) для диалогов

## Стек
- Python
- aiogram 3
- FSM (Finite State Machine)

## Запуск

```bash
pip install aiogram
python main.py
